# DOM Explorer - Interactive Task Manager

A simple Task Manager built using HTML, CSS, and Vanilla JavaScript. The goal of this project was to get hands-on experience with DOM manipulation, event handling, event propagation, and browser rendering concepts without using any frameworks or libraries.

## Live Demo

https://task-manager-madhavi.vercel.app

## Features

* Add, edit, complete, and delete tasks
* Organize tasks using categories
* Light/Dark theme toggle
* Attributes vs Properties demonstration
* Event Bubbling, Capturing, and stopPropagation demo
* Event Delegation for task actions
* Interactive educational section covering DOM concepts

## Concepts Practiced

* DOM Manipulation
* Event Handling
* Event Delegation
* Event Bubbling & Capturing
* Browser Rendering Pipeline
* Custom Data Attributes

## Tech Stack

* HTML
* CSS
* JavaScript (ES6)
